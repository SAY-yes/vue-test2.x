## Create project

* npm install -g @vue/cli @vue/cli-service-global
* vue create vue-test2.x

## Project install

* vue add route
* vue add vuex

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
